This is an implementation of edexcel pseudocode.
Based on: https://tools.withcode.uk/ks4pseudo/media/edexcel_pseudocode.pdf

Implemented:
 -Reading an input file
 -Comments
 -Tokenising the file
 -Basic parsing (Only variable declarations and assignments with integer literals)
